## 行业列表：`get_industries`

```
from jqdata import *
get_industries(name, date=None)
```

name表示行业代码：

"sw_l1": 申万一级行业
"sw_l2": 申万二级行业
"sw_l3": 申万三级行业
"jq_l1": 聚宽一级行业
"jq_l2": 聚宽二级行业
"zjw": 证监会行业
