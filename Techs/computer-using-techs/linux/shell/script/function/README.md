## 函数

函数按如下格式定义，之后调用的时候直接使用`function_name`即可。

```
function_name () { 
   list of commands
}

function_name () 
{ 
   list of commands
}

# 调用函数
function_name
# 将调用结果赋值给变量var
var=$(function_name)
```
