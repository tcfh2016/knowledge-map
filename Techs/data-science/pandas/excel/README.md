## 用pandas读取excel

在excel里面能够做的工作基本在pandas里面也可以做。pandas相比excel的优势：

- excel能够处理100万行的数据，pandas能够处理上亿行的数据
- 一些内存密集型的计算可能会把excel搞崩溃，但是pandas处理得游刃有余
- pandas能够更好的将一些处理进行自动化
- excel只能够在Windows上使用，但是pandas处理的数据是跨平台的
