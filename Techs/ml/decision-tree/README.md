## 决策树

决策树是一种通过树形结构进行分类的方法。在决策树中，树形结构中每个节点表示对分类目标在属性上的一个判断，每个分支代表基于该属性做出的一个判断，最后树形结构中每个叶子节点代表一种分类结果。

- “根节点”只有向下分类的箭头
- 除了“根节点”和“叶子节点”的为“中间节点”，“中间节点”总是有箭头指向它们，也有箭头离开它们
- “叶子节点”只有箭头指向它们，但没有箭头离开它们
