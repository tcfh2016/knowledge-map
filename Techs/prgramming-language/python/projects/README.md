### 什么是 `nltk`？

NLTK（Natural Language Toolkit）是一个用于处理和分析人类语言数据的开源库，主要用于自然语言处理（NLP）。它提供了丰富的工具和资源，帮助开发者和研究人员进行文本处理和语言数据分析。

内置词典资源，如WordNet，一个用于词汇关系分析的词典。
