## 基本常识

- Python中可以使用分号`;`将多行代码集中在一行中进行编写。
- 布尔类型的值为`True`和`False`。
- 使用`isinstance()`函数可以确认对象是否属于特定的类型或类。
- 生命多行的字符串需要使用3个单引号：`'''`。
- Python中大多数对象是可变的（Mutable）对象，但元组和字符串是不可变的（Immutable）对象。
