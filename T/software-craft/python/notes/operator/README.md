# 分类

Python操作符可以分为如下7类：

## 算术运算符

`+ - * / % ** //`

## 赋值运算符

- `=`
- `+= -= *= /= %= **= //=`
- `&= |= ^= >>= <<=`

## 比较运算符

`== != > < >= <=`

## 逻辑运算符

`and or not`

## 一致运算符

判断两个表量是否指向同一对象。

- `is`
- `is not`

## 按位运算符

`& | ^ ~ << >>`

## 成员运算符

测试序列的成员，对字符串、列表或者元组均适用。

- `in`
- `not in`
